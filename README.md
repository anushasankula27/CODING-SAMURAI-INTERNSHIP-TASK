# CODING SAMURAI INTERNSHIP TASK

This repository contains the projects completed as part of the Coding Samurai Internship Program.  
The tasks demonstrate practical implementation of Machine Learning and Natural Language Processing concepts using Python.
### Project 1: Linear Regression on Sales Data

### 📌 Objective:
To build a Linear Regression model to predict sales based on advertising spend.

### 🛠 Technologies Used:
- Python
- Pandas
- Matplotlib
- Scikit-learn

### 📊 Outcome:
Successfully built and evaluated a regression model to understand the relationship between advertising budget and sales.
### 📂 Dataset:
- Advertising Dataset
- Features: TV, Radio, Newspaper advertising spend
- Target: Sales

### 🛠 Technologies Used:
- Python
- Natural Language Processing (NLP)

### 📊 Outcome:
Built a sentiment classification model to understand customer opinions from text data.

### 📂 Dataset:
- Text dataset containing customer reviews
- Used for sentiment classification (Positive, Negative, Neutral)
